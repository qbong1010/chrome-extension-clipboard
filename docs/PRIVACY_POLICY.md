# 개인정보처리방침 (Privacy Policy)

**Paste Right** 확장 프로그램은 사용자의 개인정보를 보호하며, 아래와 같은 원칙을 따릅니다.

## 1. 수집하는 정보
본 확장 프로그램은 **어떠한 개인정보도 수집, 저장, 전송하지 않습니다.**

## 2. 권한 사용 목적
확장 프로그램의 핵심 기능을 제공하기 위해 다음 권한만을 사용합니다:
- **`contextMenus`**: 우클릭 메뉴를 통해 템플릿을 선택할 수 있도록 합니다.
- **`activeTab` 및 `scripting`**: 사용자가 현재 활성화한 웹 페이지(입력창)에 선택한 텍스트 템플릿을 붙여넣기 위해 사용합니다.
- **`storage`**: 사용자가 생성한 템플릿 목록과 자주 쓰는 문구 데이터를 사용자의 로컬 브라우저에만 안전하게 저장합니다.

## 3. 정보의 보관 및 제3자 제공 여부
사용자가 생성한 템플릿(문구)과 확장 프로그램 설정 데이터는 전적으로 사용자의 브라우저 로컬 저장소(`chrome.storage.local`)에만 보관됩니다. 외부 서버나 제3자에게 어떠한 정보도 전송하거나 공유하지 않습니다.
