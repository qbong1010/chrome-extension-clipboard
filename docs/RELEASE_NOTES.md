# Release Notes (재구성)

> 버전 태깅이 일부 구간에서 일관되지 않아, Git 커밋 히스토리와 기능 도입 시점을 기준으로 실사용 관점의 버전을 재구성했습니다. 최신 안정 버전은 **v1.5**입니다.

## v1.5 (최신)

### 핵심 추가 기능
- 건축물대장 조회 기능 추가 (공공데이터 API 연동)
- 주소 검색 기반 시군구코드/법정동코드 자동 입력
- 법정동 코드 찾기 전용 흐름(검색 화면 + 전용 페이지)으로 UX 분리
- KONEPS(나라장터) 상세조회 및 첨부파일 다운로드 링크 처리

### 주요 개선
- 설정 화면에서 API 키/저장소 정책 연동 강화
- 템플릿 저장소(local/sync) 라우팅 및 전환 안정성 개선
- 사이드패널 스크롤, 카드 축소, CSP 관련 오류 수정
- E2E 테스트(Playwright) 및 저장소 동작 테스트 추가

### 사용자 체감 변화
- 단순 템플릿 도구에서 **"템플릿 + 공공데이터 조회"** 확장형 도구로 진화
- 실무 입력값(코드/주소) 탐색과 템플릿 활용이 한 화면 흐름으로 연결됨

## 버전 재구성 기준

- 명시 버전(예: `v1.4`, `v1.5`)이 있는 커밋 우선 반영
- 버전 표기가 누락된 구간은 기능 단위(대규모 UI 개편, 클립보드 도입, 설정/저장소 개편, 공공 API 확장)로 묶어 재구성
- 팀 내/스토어 배포 시에는 이후부터 `MAJOR.MINOR.PATCH` 고정 규칙 사용 권장
