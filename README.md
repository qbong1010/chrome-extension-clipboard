# Paste Right Manager

크롬 확장 프로그램으로 제작된 반복 사용 메시지 템플릿 관리 및 건축물대장 조회 도구입니다.

## 주요 기능

### 📋 템플릿 관리
- 자주 쓰는 메시지 템플릿 생성 및 관리
- 우클릭 메뉴로 빠른 붙여넣기
- 클립보드 복사 버튼 (원클릭 복사)
- 드래그 앤 드롭으로 순서 변경
- Material Design 3 스타일

### 🏢 건축물대장 조회
- 시군구코드, 법정동코드, 지번으로 건축물 정보 조회
- 공공데이터 API 연동
- 간략 보기 / 상세 보기 전환
- 주요 정보: 건물명, 주소, 용도, 구조, 면적, 층수 등

### 🔍 주소 검색
- 전국 20,555개 법정동 주소 검색
- 주소 입력으로 코드 자동 조회
- 검색 결과 선택 시 코드 자동 입력
- 띄어쓰기 무시 검색 지원

## 설치 방법

### 개발자 모드 설치
1. <kbd>Code</kbd> → <kbd>Download ZIP</kbd> 클릭하여 다운로드
2. Chrome 브라우저 주소창에 입력:
   ```
   chrome://extensions
   ```
3. 우측 상단 **개발자 모드** 활성화
4. **압축해제된 확장 프로그램을 로드합니다** 클릭
5. 압축 해제한 폴더 선택

### 의존성 설치 (개발자용)
```bash
npm install
```

## 사용 방법

### 템플릿 관리
1. Chrome 툴바에서 확장 프로그램 아이콘 클릭
2. 사이드 패널이 열림
3. 템플릿 헤더 우측의 **+** 버튼으로 새 템플릿 추가
4. 템플릿 항목의 **복사 아이콘**을 클릭하여 클립보드에 복사
5. 우클릭 메뉴에서도 템플릿 선택 가능

### 건축물대장 조회
1. 사이드 패널에서 **검색 아이콘(🔍)** 클릭
2. **주소로 코드 찾기** 섹션에서 주소 검색:
   - 예: "경기도 남양주시 금곡동"
   - 검색 결과에서 "선택" 클릭
   - 코드가 자동으로 입력됨
3. 본번(4자리), 부번(4자리) 입력
   - 예: 685, 17 (자동으로 0685, 0017로 변환)
4. **조회** 버튼 클릭
5. 간략 보기 / 상세 보기 전환하여 정보 확인

## 기술 스택

- **Frontend:** HTML5, CSS3, JavaScript ES6+
- **Design:** Material Design 3
- **Libraries:** Sortable.js, Material Symbols
- **APIs:** Chrome Extension API, 공공데이터포털 API
- **Data:** 전국 법정동코드 20,555개
